# Links em MarkDown

Podemos adicionar links de 2 maneiras, em um texto (Hyperlink), e somente tornarmos um link "clicavél"

Para transformar um texto em link clicável, usamos:

"[texto] (link)"
* Obs: Sem o espaço
Exemplo:
[google](google.com)

e para deixarmos um link ser funcional, usamos:

"<>"
<google.com>

# Tabelas

Exemplo 2 | R$ 5,00

Exemplo 3 | R$ 5,50
